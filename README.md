# Service Router Operator

A Kubernetes operator to automate multi-region DNS provisioning on AKS using Istio and ExternalDNS. Built with [Kubebuilder](https://github.com/kubernetes-sigs/kubebuilder).


## Overview

For a detailed explanation of how this operator was built and how it works, check out my [Service Router Operator blog post](https://teknologi.nl/posts/servicerouteroperator).
